# Simple Redirect Page

A single-file HTML landing page that automatically redirects visitors using JavaScript.

## Code Overview
* **Active Route:** Redirects to a WhatsApp link with a pre-filled inquiry message (`wa.me`).
* **Alternative Route:** Includes a commented-out backup destination (`activetrail.biz`).

## Setup
1. Upload `index.html` to a GitHub repository.
2. Enable **GitHub Pages** in the repository settings to host it as a live redirect link.
